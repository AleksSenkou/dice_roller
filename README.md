### Task description:
- calculate the probability of rolling a given amount on dices
- for 100 dices time of code execution should be less then 1 sec

#### Conditions:
- N - number of dices
- M - amount

#### Examples:
- N = 2, M = 1 => P = 0
- N = 2, M = 2 => P = 1/36
- N = 5, M = 9 => P = 70/7776
